Application Setup :

    Client :

```
git clone https://github.com/codewithashim/todo-frontend.git 
```

```
cd todo-frontend
```

```
npm install
```

```
npm run dev
```

   Server :

```
git clone https://github.com/codewithashim/todo-backend.git 
```

```
cd todo-backend
```

```
yarn install
```

```
yarn dev
```
